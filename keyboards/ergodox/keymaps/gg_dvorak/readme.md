I was a long time TypeMatrix 2030 user with the Dvorak layout. This layout
takes some cues from the TM2030. I've also made some tweaks for usage with
the Object Pascal IDEs I work with daily. Some keystrokes still don't work
well with those IDEs, so expect a few more revisions in the future.

Graeme Geldenhuys.
